# API development with Express.js and Node.js

1. Put and complete the info below in a .env file:

```
# GENERAL CONFIGURATION
PORT =
MORGAN_MODE = 'dev'

# DATABASE DATA INFO
MONGODB_URI =
MONGODB_DB =

# ENCRYPTION
ENC_SALT_ROUNDS =

# JWT
JWT_CLAVE =
JWT_EXPIRES =
```

2. Install the dependencies to the local node_modules folder:

```npm i```
